# PinPig 앱 컨셉

---

## 핵심 정의

**PinPig** - 소비를 통해 관심을 비춰주는 거울

> "기록하는 가계부가 아니라, 비춰주는 거울"

---

## 핵심 질문의 전환

| 기존 가계부 | PinPig |
|------------|--------|
| "얼마나 썼지?" | **"얼마나 남았지?"** |
| 지출 = 죄책감 | 소비 = 관심의 표현 |
| 과거 기록 | 현재 상태 |
| 분석 중심 | 확인 중심 |

---

## 타겟 사용자

### 페르소나: 직장인

**특징:**
- 월급이라는 고정 수입
- 매달 반복되는 지출 패턴
- "이번 달 얼마나 남았지?"가 핵심 관심사
- 복잡한 자산관리보다 단순 확인 원함

**니즈:**
- 월급 받고 → 고정지출 빼고 → 남은 돈 확인
- 오늘 써도 되는 돈이 얼마인지
- 내가 뭐에 관심 있는지 (소비 = 관심)

---

## 차별화 포인트

### 메인 기능: 소비계획 & 소비항목 확인

```
┌─────────────────────────────────────┐
│                                     │
│         847,000원                   │
│      이번 달 쓸 수 있는 돈           │
│                                     │
│   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━     │
│                                     │
│      하루 56,000원 남음             │
│                                     │
└─────────────────────────────────────┘
```

**핵심:**
1. **소비계획** - 예산 설정 → 남은 돈 실시간 확인
2. **소비항목** - 내가 뭐에 돈을 쓰는지 = 뭐에 관심 있는지

## 톤 & 보이스

### 조용한 관찰자

| 원칙 | 예시 |
|------|------|
| 판단하지 않음 | ~~"너무 많이 썼어요"~~ → "예산의 80%를 사용했어요" |
| 사실만 전달 | ~~"절약하세요!"~~ → "하루 56,000원 남았어요" |
| 관심 비춰줌 | ~~"식비 초과"~~ → "이번 달 식비에 가장 많이 썼어요" |

**핵심 어조:**
- 중립적
- 담담한
- 따뜻한 (차갑지 않은)

## 앱 이름 의미

### PinPig (핀피그)

**해석 1:** Pin (꽂다) + Pig (돼지저금통)
- 동전을 꽂는 돼지저금통
- 저축/관리의 상징

**해석 2:** Pin (핀, 표시) + Pig (돼지)
- 소비에 핀을 꽂아 표시
- 관심사를 기록

---

## 경쟁사 vs PinPig

| 항목 | 일반 가계부 | PinPig |
|------|------------|--------|
| 초점 | 지출 기록 | 남은 돈 확인 |
| 감정 | 죄책감, 압박 | 중립, 편안 |
| 복잡도 | 많은 기능 | 필수 기능만 |
| 디자인 | 금융앱 느낌 | 노트앱 느낌 |
| 목표 | 절약 | 인식 (awareness) |

## 핵심 경험 흐름

```
[앱 열기]
    ↓
[남은 돈 즉시 확인] ← 1초 안에
    ↓
[오늘 쓸 수 있는 돈 확인]
    ↓
[필요시 기록 추가] ← 3터치 이내
    ↓
[월말: 내 관심사 확인]
```

---

## 설계 원칙

1. **1초 확인** - 앱 열면 바로 "남은 돈" 보임
2. **3터치 기록** - FAB → 금액+카테고리 → 저장
3. **판단 없음** - 사실만 전달, 평가하지 않음
4. **관심 비춤** - 소비 = 관심, 카테고리로 확인
