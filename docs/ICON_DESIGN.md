# PinPig 앱 아이콘 디자인 명세

## 디자인 철학

> **"기록하는 가계부가 아니라, 비춰주는 거울"**

PinPig 아이콘은 금융앱의 차갑고 딱딱한 느낌이 아닌,
일상에서 매일 쓰는 노트앱처럼 따뜻하고 친근한 인상을 줘야 합니다.

---

## 아이콘 콘셉트: "Minimal Piggy"

### 핵심 요소
```
1. 돼지코 (Pig Snout)
   - 두 개의 원형 콧구멍
   - 심플하고 귀여운 형태
   - 돼지 저금통을 상징

2. 코인 슬롯 (Coin Slot)
   - 돼지코 위의 가로선
   - 저축/기록의 상징
   - 선택적 요소 (간결함 우선 시 생략 가능)
```

### 시각적 표현
```
┌─────────────────────────┐
│                         │
│       ┌───────┐         │
│       │  ═══  │  ← 코인 슬롯 (선택)
│       └───────┘         │
│                         │
│      ╭─────────╮        │
│     │  ●   ●  │        │  ← 돼지코 (콧구멍 2개)
│      ╰─────────╯        │
│                         │
│                         │
└─────────────────────────┘
```

## 사이즈 가이드

### 필수 사이즈
```
192x192px  - Android 홈화면, PWA 기본
512x512px  - 스플래시 스크린, 고해상도
```

### 안전 영역 (Safe Area)
```
전체 크기의 10% 마진 유지
192px → 아이콘 영역 154px, 마진 19px
512px → 아이콘 영역 410px, 마진 51px
```

### 모서리 반경
```
둥근 모서리 적용 안 함 (OS가 자동 처리)
내부 요소는 부드러운 곡선 사용
```

## 디자인 원칙

### DO ✅
- 단순하고 인식하기 쉽게
- 따뜻한 톤의 색상 사용
- 작은 크기에서도 명확하게
- 여백을 충분히 확보

### DON'T ❌
- 복잡한 디테일 추가
- 차가운 파란색/회색 사용
- 텍스트 포함
- 그라데이션 과다 사용

## 예상 결과물

### 라이트 버전 (기본)
```
배경: #FAF9F7 (따뜻한 화이트)
돼지코: #3D3C3A (잉크 다크)
콧구멍: #FAF9F7 (배경색, 구멍 표현)
전체 인상: 깔끔하고 미니멀한 저금통
```

### 특징
- 멀리서 봐도 "돼지"임을 알 수 있음
- 금융앱 느낌 없이 친근함
- iOS/Android 모두 어울림
- 다크모드 대응 가능
