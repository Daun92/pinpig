# PinPig 사용자 경험 여정

---

## 핵심 플로우

### 1. 메인 플로우: 남은 돈 확인

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   [앱 열기] ──1초──→ [남은 돈 확인] ──→ [앱 닫기]            │
│                                                             │
│   가장 자주 발생하는 플로우. 입력 없이 확인만.                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**빈도:** 하루 3-5회
**소요시간:** 3초 이내
**핵심:** 아무 액션 없이 정보 획득

---

### 2. 거래 기록 플로우

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   [앱 열기] → [FAB 탭] → [금액 입력] → [카테고리] → [저장]   │
│       0        1           2            2           3       │
│                                                             │
│   3터치 이내 완료. 평균 10초.                                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**빈도:** 하루 2-5회
**소요시간:** 10초 이내
**핵심:** 최소 입력, 스마트 기본값

---

### 3. 내역 확인 플로우

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   [홈] → [모두보기] or [내역탭] → [스크롤] → [상세확인]      │
│                                                             │
│   "어제 뭐 샀더라?" 궁금할 때                                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**빈도:** 하루 0-2회
**핵심:** 날짜별 그룹핑으로 빠른 탐색

---

### 4. 월말 리뷰 플로우

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   [통계탭] → [카테고리별 확인] → [관심사 인사이트]           │
│                                                             │
│   "이번 달 뭐에 관심 있었지?"                                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**빈도:** 월 1-2회
**핵심:** 소비 = 관심, 판단 없는 비춤

## 핵심 경험 원칙

### 1. 1초 확인

```
앱 열기 → 남은 돈 보임
         (추가 액션 없음)
```

- Hero 영역에 "남은 돈"
- 스크롤 없이 핵심 정보
- 로딩 없이 즉시 표시 (로컬 DB)

### 2. 3터치 기록

```
FAB(1) → 금액+카테고리(2) → 저장(3)
```

- 숫자패드 기본 표시
- 시간대별 카테고리 추천
- 날짜는 "오늘" 기본값

### 3. 판단 없는 비춤

```
❌ "지출이 많습니다"
⭕ "예산의 80%를 사용했어요"

❌ "카페를 줄이세요"
⭕ "카페에 관심이 많았어요"
```

- 중립적 어조
- 사실만 전달
- 판단은 사용자 몫

### 4. 관심 = 소비

```
기존 가계부: 지출 = 죄책감
PinPig:     소비 = 관심 표현
```

- "뭐에 돈을 많이 썼나" = "뭐에 관심이 많았나"
- 부정적 프레이밍 제거

## 감정 곡선

### 일반 가계부 앱

```
감정
 ↑
 │     설정복잡
 │    ↗        ↘
시작 ─            ─→ "또 초과했네" → 삭제
 │                  ↘
 ↓                    죄책감
```

### PinPig

```
감정
 ↑
 │  간단!    편하네    그렇구나
 │   ↗         ↗        ↗
시작 ─────────────────────→ 지속 사용
 │
 ↓
```

**핵심:** 부정적 감정 최소화, 중립적 정보 제공
